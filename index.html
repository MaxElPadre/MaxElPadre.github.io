C# est un langage phare du framework .net qui se popularise pour la conception de sites Web (ASP), d’ERP (Sharepoint), de scripting et d’applications lourdes. Appelé « petit frère » du Java car il en reprend la syntaxe générale ainsi que les concepts. le C# bénéficie d’une politique de développement très dynamique.

Aujourd’hui, C# est un langage de programmation moderne et très utilisé avec une bibliothèque standard très fournie et des outils de développement avancés.

JavaScript

Créé en 1995 par Brendan Eich, JavaScript désigne un langage de script orienté objet. On le retrouve principalement dans les pages Internet. Il est principalement employé pour améliorer l'ergonomie d'un site Internet et/ou d'une interface applicative utilisateur. Il permet également, entre autres, d'introduire sur une page web ou HTML des animations ou des effets (animations 2D/3D, pop-up, afficher du contenu mis à jour à des temps déterminés, des cartes interactives, etc).

JavaScript offre en effet la possibilité d'exécuter un code sans être contraint de recharger une page web. En cela, il joue un rôle dans l'amélioration de la rapidité de chargement des pages, un critère d'ergonomie et de référencement de plus en plus important.

Le langage JavaScript se distingue des langages serveurs par le fait que l'exécution des tâches est opérée par le navigateur lui-même, sur l'ordinateur de l'utilisateur, et non sur le serveur web. C’est la troisième couche des technologies standards du web avec HTML et CSS

SQL

Créé au début des années 1970 par Donald D. Chamberlin et Raymond F. Boyce, tous deux chez IBM, pour pouvoir manipuler simplement les données.

Le langage SQL (Structured Query Language) est un langage informatique utilisé pour exploiter des bases de données. Il est dérivé de l'algèbre relationnel et de SEQUEL. Il permet, entre autres, de pouvoir manipuler, éditer, traiter des données stockées dans une base de données dite relationnelle. La première version commerciale fut disponible en 1979.

Normalisé depuis 1986, le langage est reconnu par la grande majorité des systèmes de gestion de bases de données relationnelles (SGBDR) du marché. Le SQL est reconnu comme une norme internationale.

Aujourd’hui, de nombreuses bases de données s'appuient sur le langage SQL. C'est le cas de MySQL qui fait partie de la suite de logiciels libres LAMP, mais aussi des serveurs de base de données Oracle, D2B, Microsoft SQL Server, etc.

Savoir effectuer des requêtes n’est pas trop difficile, mais il convient de véritablement comprendre comment fonctionne le stockage des données et la façon dont elles sont lues pour optimiser les performances.

Travail En Équipe

"Aucun de nous ne sait ce que nous savons tous, ensemble." Euripide, dramaturge

Aujourd'hui avec l'accélération du rythme des projets et des technologies, avec le développement des organisations agiles, il est impossible de tout savoir et de tout savoir-faire. C'est cette cohésion d'équipe qui conduit les collaborateurs à coopérer pour réaliser le projet qui leur est confié.

Travailler en équipe me permet de favoriser le développement de mes compétences et de celles de mon entourage professionnel. J'ai pu constater également que cela limite les risques en cas d'absence et de développer l'intelligence collective.
